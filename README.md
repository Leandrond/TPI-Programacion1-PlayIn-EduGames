# Play.In EduGames - Proyecto de Plataforma Educativa

Este proyecto es el **Trabajo Práctico Integrador (TPI)** correspondiente a la asignatura **Programación I**. 

## 📝 Descripción General
Desarrollado a pedido de la empresa **Play.In**, el sistema consiste en una plataforma educativa interactiva destinada a estudiantes de **nivel primario**. La aplicación integra diversos juegos y herramientas didácticas dentro de un único sistema, permitiendo que el usuario aprenda mientras juega.

El programa cuenta con un **menú principal** que permite:
* Acceder a las diferentes actividades desarrolladas por el equipo.
* Regresar al menú principal desde cualquier punto de la aplicación.
* Finalizar la ejecución de manera segura.

## 🚀 Características Técnicas
Para cumplir con los requisitos mínimos obligatorios de la cátedra, el proyecto implementa:
* **Modularización:** División del código en funciones definidas por el programador para evitar la repetición y facilitar la reutilización.
* **Estructuras de Control:** Uso de lógica secuencial, selectiva y repetitiva.
* **Estructuras de Datos:** Manejo de cadenas de caracteres y estructuras avanzadas (listas, matrices o diccionarios) debidamente justificadas.
* **Persistencia:** Almacenamiento de datos (como rankings o puntajes) mediante **archivos de texto**.
* **Control de Versiones:** Desarrollo colaborativo y progresivo utilizando **Git**.
